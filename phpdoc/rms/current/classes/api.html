    Cleanse the given input string for use in an SQL query.

    cleanse(string $input, boolean $html, boolean $sprintf) : string
    Static

    This will perform things such as escape character checks and HTML scrubbing (if enabled). By default, the string is formated for use in an sprintf static function (i.e., percetange signs are esacped by '%%').

    Parameters

    $input

    string

    The input string to cleanse

    $html

    boolean

    If HTML scrubbing (conversion to HTML entities) should be performed (default = true)

    $sprintf

    boolean

    If percentage signs should be esacped for use in a sprintf static function (default = true)

    Returns

    stringThe cleansed string

    Creates a default 200 state in an array.

    create_200_state(object $data) : array
    Static

    This includes a 'true' in the 'ok' element, 'Success.' in the 'msg' element, and $data in the 'data' element. The header is also changed to the 200 state.

    Parameters

    $data

    object

    The data to place in the 'data' element of the array

    Returns

    arrayThe filled array

    Creates a default 401 state in an array.

    create_401_state() : array
    Static

    This includes a 'false' in the 'ok' element, 'Insufficient authorization.' in the 'msg' element, and null in the 'data' element. The header is also changed to the 401 state.

    Returns

    arrayThe filled array

    Static
