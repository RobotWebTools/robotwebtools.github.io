Announcements from Robot Web Tools
March 31, 2013

New Website Launched

The Robot Web Tools organization is proud to announce its new website. Along with bringing you the latests news related to the effort, this site will now showcase some demos and examples of the tools in action! And, as always, we welcome your contributions!

March 29, 2013

CDN Servers are Up

In an effort to make using the tools provided by Robot Web Tools, we are excited to provide CDN services for most of our libraries. In addition to uploading different revisions of our tools and their dependencies, we are also providing resources such as robot Collada models!

March 20, 2013

Standarized Libraries Started

The core functionality of the Robot Web Tools JavaScript tools are being broken up into three libraries. roslibjs will be responsible for the core communication to ROS via rosbridge, ros2djs will contain modules for creating 2D visualization widgets, and ros3djs will contain modules for creating 3D visualization widgets.
